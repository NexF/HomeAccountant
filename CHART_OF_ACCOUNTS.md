## 编码规则

| 前缀 | 类型 | 说明 |
|------|------|------|
| `1xxx` | 资产 | 货币资金、现金等价物、投资、固定资产等 |
| `2xxx` | 负债 | 信用卡、借款、应付款等 |
| `3xxx` | 权益 | 期初净资产、本期损益 |
| `4xxx` | 收入 | 工资、副业、投资收益等 |
| `5xxx` | 费用 | 衣食住行、医疗、教育等 |

- 一级科目：4 位数字编码
- 二级子科目：`父编码-序号`（如 `1001-01`）
- 三级子科目：`父编码+序号`（如 `1001-0201`）
- 用户可通过 API 自行添加自定义科目（`is_system=False`）
